# Docker, NGINX, PHP-FPM, PostgreSQL, Symfony
Docker infrastructure for symfony web aplication
build:
```
cd docker
docker-compose up --build
```
up:
```
cd docker
docker-compose up
```
down:
```
cd docker
docker-compose down
```
./app - its project files folder
